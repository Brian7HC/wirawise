# WIRA WISE - Mũndũ wa Murimi

Kikuyu Agricultural Assistant - A voice-enabled chatbot for Kikuyu farmers.

## Features

- **Text Chat**: Ask questions in Kikuyu or English
- **Voice Input**: Speak directly to the chatbot
- **Agriculture Knowledge**: Get answers about crops, farming, pests, and more
- **Bilingual Support**: Works in Kikuyu (Gĩgĩkũyũ) and English

## Tech Stack

- **Backend**: FastAPI, PostgreSQL, LangChain, Groq
- **Frontend**: Vanilla HTML/CSS/JS
- **Speech**: Meta MMS (Multilingual Speech Model)
- **NLP**: Intent classification, RAG, Translation

## Quick Start

```bash
# Install dependencies
pip install -r backend/requirements.txt

# Run the server
cd backend && python main.py
```

Open http://localhost:8000 in your browser.
